# Интсрукция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивом, нужно обрамить его звёздочкой (*). Например *вот так*.

Чтобы выделить текст полужирным, необходимо обрамить его двумя звёздочками(**). Например, **вот так**.
